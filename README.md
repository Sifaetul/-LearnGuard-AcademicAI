# 🎓 LearnGuard AI — Academic Intelligence Platform

An AI-powered academic management system built for universities to monitor student performance, detect at-risk students early, and provide intelligent learning assistance.

## 🚀 Features

- **📊 Dashboard** — Real-time student attendance, CGPA, and department analytics
- **📚 My Courses** — Course details, assignments, labs, and exam schedules
- **🤖 BLC Chatbot** — AI-powered academic assistant for student queries
- **⚠️ Early Warning System** — Detects at-risk students based on performance data

## 🛠️ Tech Stack

**Frontend**
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Axios

**Backend**
- Python (FastAPI)
- Groq AI API
- REST API

## ⚙️ Setup

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## 🌐 Running

- Frontend: http://localhost:3000
- Backend: http://localhost:8000
